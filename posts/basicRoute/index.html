Basic Route

Dec 17, 2020 2020-12-17T16:30:00+09:00 by SeonjaeLee
1 min

기본라우팅

URI(또는 경로) 및 특정한 HTTP 요청 메소드(GET, POST 등)인 특정 엔드포인트에 대한 클라이언트 요청에 애플리케이션이 응답하는 방법을 결정하는 것을 말합니다
각 라우트는 하나 이상의 핸들러 함수를 가질 수 있으며, 이러한 함수는 라우트가 일치할 때 실행됩니다

1

app.METHOD(PATH,HANDLER);

METHOD => HTTP 요청 메소드(get, post, put, send)
PATH => 경로 지정 / 만약 path안에 $를 넣고 싶으면 ([\$])라고 해야됨
HANDLER => 일치할때 실행되는 함수

//Example root/app.js
let express = require('express')
let app = express()

app.get('/',(req,res)=>{
  res.send('hi')
})

app.listen(3000,(req,res)=>{
    console.log('ok');
})
