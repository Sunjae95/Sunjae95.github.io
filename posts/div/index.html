CSS - 화면정리

Feb 3 2021-02-03T13:23:00+09:00 by SeonjaeLee
3 min

DIV CSS

Naver API를 이용한 애플리케이션을 만들던 도중 디자인을 네이버 길 찾기와 비슷하게 해야겠다 생각해서 무작정 CSS를 공부했다.
bootstrap, reactstrap도 사용해봤지만 내가 원하는 것은 버튼을 꾸미고 input 칸을 꾸미는 것이 아니라 layout을 설정하는 것이 목표였기에 못 찾게 되었다.
그러다가 CSS를 다시 공부해보자 해서 내가 원하는 layout이 되었다.

    <div className="Search">

결과


결과는 세로로 나왔다.
나는 세로가 아닌 가로로 정렬되어 있는 것을 원했기에 CSS를 공부하고 다음과 같이 코드를 짰다.

CSS code

div {
    height: 100vh;
    background-color: black;
}
.Search {
    width: 10%;
    height: 100%;
    background-color: blue;
    display: inline-block;
}
.Map {
    width: 90%;
    height: 100%;
    background-color: blueviolet;
    display: inline-block;
}

결과

크게 div로 묶어 높이단위를 vh를 사용하여 화면을 꽉차게 만들었다.
그리고 div는 입력이되면 칸을 띄어 출력이 되지만 display: inline-block을 적용하여 칸을 띄우지 않고 가로로 출력되게 해준것이다.

생각보다 간단했지만 CSS를 공부하다 보면 같게 보이지만 다르게 표현할 수도 있기 때문에 내가 원하는 방식으로 만들기까지 시간이 조금 걸렸다.
혹시라도 잘못됐다면 이 포스트를 고치도록 하겠다. 지금 상태에서는 나의 최선의 방법이라고 생각한다.

틀린것 고치기

1
2
3
4

div {
    height: 100vh;
    background-color: black;
}

위의 같은 경우 div를 한번에 선언했지만 내가 만든 어플리케이션은 내가 원하는 이미지로 안나왔다. 그래서 전체 div로 묶지 않고 클래스를 주어 다르게 묶어봤다.

.main {
    height: 100vh;
    background-color: black;
}
.search {
    width: 15%;
    height: 100%;
    background-color: blue;
    float: left;
}
.map {
    width: 85%;
    height: 100%;
    background-color: blueviolet;
    float: left;
}

div 전체를 묶지 않고 각 클래스 별로 주어 main안에 search와 map을 포함시켜 크기는 main을 따르게 하고 float를 활용하여 수평으로 정렬해주었다. (display: inline-block과 같은 방법)

결과

이렇게 검색창과 지도창을 나누어 사용자가 보기 좋게 만들었다. float나 display 속성을 활용하여 layout을 만드는것도 좋지만 다음에는 flex를 이용하여 css를 다뤄보는것도 좋다고 생각이 들었다.
