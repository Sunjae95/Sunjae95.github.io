### Promise
`promise`란 비동기적 프로그램에서 콜백함수대신 쓰는 javascript object이다.  

1. 현재 상태를 확인할 것  
(pending(오퍼레이션이 실행 중일 때) -> fulfilled(오퍼레이션을 성공적을 끝냈을 때) or rejected(실패 했을 때))
2. producer와 comsumer의 차이점을 확인하는 것이 중요  
  
### Producer
```javascript
const promise = new Promise((resolve, reject) => {  //executor콜백함수 자동생성
    //doing some heavy work (network, read files 등..)
    console.log('doing something...');
    setTimeout(() => {
        //resolve('seonjae');   성공적일때
        rejec t(new Error('no network'));    //실패시
    }, 2000);
})
//Promise를 생성한 순간 executor가 실행되기 때문에 동작을 하고 움직이는 활동을 한다면 주의해서 사용해야된다.
```  
### Consumers
```javascript 
//위의 promise에서 성공, 실패를 정의를 하고 then과 catch를 이용해 알려준다.
promise
    .then((value) => {  //성공시
    console.log(value);
    })
    .catch(error => {   //실패시
        console.log(error);
    })
    .finally(() => {    //성공 실패 상관없이 무조건 호출
        console.log('finally');
    });
```  
  
### Promise chaining
```javascript
const fetchNumber = new Promise((resolve, reject) => {
    setTimeout(() => resolve(1), 1000);
});

fetchNumber
.then(num => num * 2)   //num에 1 전달
.then(num => num * 3)   //num에 2 전달
.then(num => {          //num에 6 전달
    return new Promise((resolve, reject) => {
        setTimeout(() => resolve(num - 1), 1000);
    });
})
.then(num => console.log(num)); //1초후 num에 5전달 후 출력
//then은 값 or Promise 전달 가능
```  
  
### Error Handling
```javascript
const getHen = () => 
    new Promise((resolve, reject) => {
        setTimeout(() => resolve('🐓'), 1000);
    });
const getEgg = hen =>
    new Promise((resolve, reject) =>{
        //둘중 하나를 주석해서 실행 시킬 것!
        setTimeout(() => resolve(`${hen} => 🥚`), 1000);
        setTimeout(() => reject(new Error(`error! ${hen} => 🥚`)), 1000);
    });
const cook = egg =>
    new Promise((resolve, reject) => {
        setTimeout(() => resolve(`${egg} => 🍳`), 1000);
    });

getHen()
    .then(hen => getEgg(hen))   //value가 같으면 then안에 value를 getEgg로 생략이 가능하다. 나머지 then은 생략했음
    //resolve에  🐓을 value로 보냄
    .catch(error => {   //error가 난다면 value를 🥖으로 만들어주어 코드가 계속 실행되게 메꿔줌
        return '🥖';
    })
    .then(cook) //🐓를 받아서 resolve를 🐓 => 🥚로 저장
    .then(console.log); //🐓 => 🥚받아서 resolve에 🐓 => 🥚 => 🍳저장
    // .catch(console.log) 어디서 error 났는지 확인할 수 있음
//결과 🐓 => 🥚 => 🍳
```
