# K번째 수
### 문제
배열 array의 i번째 숫자부터 j번째 숫자까지 자르고 정렬했을 때, k번째에 있는 수를 구하려 합니다.

예를 들어 array가 [1, 5, 2, 6, 3, 7, 4], i = 2, j = 5, k = 3이라면

1. array의 2번째부터 5번째까지 자르면 [5, 2, 6, 3]입니다.
2. 1에서 나온 배열을 정렬하면 [2, 3, 5, 6]입니다.
3. 2에서 나온 배열의 3번째 숫자는 5입니다.  

배열 array, [i, j, k]를 원소로 가진 2차원 배열 commands가 매개변수로 주어질 때, commands의 모든 원소에 대해 앞서 설명한 연산을 적용했을 때 나온 결과를 배열에 담아 return 하도록 solution 함수를 작성해주세요.
  
### 제한사항
- array의 길이는 1 이상 100 이하입니다.
- array의 각 원소는 1 이상 100 이하입니다.
- commands의 길이는 1 이상 50 이하입니다.
- commands의 각 원소는 길이가 3입니다.

|**array**|**commands**|**return**|
|:-:|:-:|:-:|
|[1, 5, 2, 6, 3, 7, 4]|[[2, 5, 3], [4, 4, 1], [1, 7, 3]]|[5, 6, 3]|
  

### 입출력 예 설명
[1, 5, 2, 6, 3, 7, 4]를 2번째부터 5번째까지 자른 후 정렬합니다. [2, 3, 5, 6]의 세 번째 숫자는 5입니다.  
[1, 5, 2, 6, 3, 7, 4]를 4번째부터 4번째까지 자른 후 정렬합니다. [6]의 첫 번째 숫자는 6입니다.  
[1, 5, 2, 6, 3, 7, 4]를 1번째부터 7번째까지 자릅니다. [1, 2, 3, 4, 5, 6, 7]의 세 번째 숫자는 3입니다.

# my solution
```javascript
function solution(array, commands) {
    var answer = [];
  commands.forEach(element => {
        const tmp =array
                .slice(element[0]-1,element[1])
                .sort((a,b) =>{
                    return a-b;
                });
        answer.push(tmp[element[2]-1]);
    });
    return answer;
}
```
나는 이 문제를 보고 각 테스트케이스에 대하여 배열을 나누고 정렬해서 k 번째 인덱스를 뽑으면 된다고 생각했다. 그래서 예전에 Ellie의 드림 코딩 영상에서 나온 배열 API를 활용하였다.  
배열을 나눌 때는 `slice()`를 사용했고, 정렬할 때는 `sort()`를 사용했다. 뽑을 때는 배열 인덱스를 찾아 뽑으면 하나의 테스트케이스의 정답이 된다.  
하지만 문제에서는 테스트케이스는 여러 개가 주어지므로 한 개의 테스트케이스마다 배열의 인덱스가 되므로 `foreach()`를 사용하여 각 테스트케이스에 대하여 위와 같이 분류하고 `answer`에 값을 `push`했다. 다 맞을 것이라고 생각하고 제출을 했지만 87점을 맞았다.

나와 유사한 답을 쓴 사람이 있는데 그 사람도 나와 같은 질문을 했다. 그에 해당하는 답변은 `sort()`에 함수를 제대로 정의해야 된다고 했다.
그래서 나는 `sort()`함수를 제대로 정의하는 부분을 찾아보니
```javascript
sort((a,b) =>{
     return a-b;
});
```
이렇게 해야 오름차순으로 정렬되는것을 알수있었다.
